# AI-README

This file explains what was generated by the AI agent and how to finish installation.

## What was added
- Electron app with a custom `gbe://` protocol.
- Create page at `gbe://create` to map a local folder to a `gbe://name` domain.
- Windows installer configuration using `electron-builder` (NSIS).

## Build an installer
```
cd D:\gbe\app
npm install
npm run dist
```

## Result
The installer is created in `D:\gbe\app\dist` and behaves like a normal Windows app:
- Installs to `Program Files` or a folder you choose.
- Creates Start Menu and Desktop shortcuts.

## Notes
- `gbe://` sites run with Node.js enabled. Normal sites do not.
- If you want icons, add them to `D:\gbe\app\build` and set them in `package.json`.
